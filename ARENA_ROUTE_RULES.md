# Правила арены

Время запуска: каждый час в `:10` по московскому времени.

Маршрут:

1. Отправить `/start`.
2. Нажать `Арена (PvP)`.
3. Нажать `Рандомный бой`.
4. После результата нажимать `Ещё бой`.
5. Всего провести 5 боёв либо остановиться при сообщении о лимите.
6. После завершения обязательно отправить `/start`.
7. Вернуться к обычному фарму: `Исследовать` → выбранный этаж → последняя локация → продолжить исследование.

После окончания гильдейского маршрута обычный фарм до арены не запускать.

Без отдельного прямого указания пользователя нельзя менять время запуска `:10`, количество боёв или маршрут арены.
