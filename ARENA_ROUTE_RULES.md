# Правила арены

Время запуска: каждый час в `:02` по московскому времени.

Маршрут:

1. Отправить `/start`.
2. Нажать `Арена (PvP)`.
3. Нажать `Рандомный бой` для первого боя.
4. После результата нажимать именно `Ещё бой`.
5. Повторять `Ещё бой`, пока не будет проведено 5 боёв либо пока игра не сообщит `Рандомных боев использовано 5/5` или о достижении лимита.
6. После завершения обязательно отправить `/start`.
7. Вернуться к обычному фарму: `Исследовать` → выбранный этаж → последняя локация → продолжить исследование.

Кнопку `Рандомный бой` использовать только для первого боя. Все последующие бои запускаются кнопкой `Ещё бой`.

Без отдельного прямого указания пользователя нельзя менять время запуска `:02`, количество боёв или маршрут арены.
